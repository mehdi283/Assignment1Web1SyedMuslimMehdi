Basics of HTML Tags

<h1> to <h6> Headings

Used to Describe HTML headings. <h1> Is the Largest, <h6> Is the Smallest.

<h1>Main heading</h1>

<p> Paragraph

Captures a group of words

<p>This is a Paragraph</p>

<a> Anchor

Generates a link to a separate page or URL

<a href="https://www.google.com/">Visit Google</a>

<img> Image

Img tag displays an image in webpage

<img src="image.jpg" alt="A description" />

<ul>, <ol>, <li> Lists

Defines ordered (numbered) and unordered (bulleted) lists.


<ul>
<li>First item</li>
<li>Second item</li>
</ul>

<div> Division

A Container collects all the other words togather

<div class="box">Content here</div>

<br> Line Break

Makes one line break in the text

Line one<br>Line two

<strong> and <em> Emphasis

<strong> Style the code Bold <em> Capitalize it.

<strong>Bold Text</strong> and <em>Italic Text</em>

Basics of CSS

color

Sets the Font color

color: Red;

Background-color

Adjust the background-color of an element

background-color: #fff;

font-family

Selects the Font style of the text

font-family: 'Segoe UI', sans-serif;

padding

Sets the space inside the text

padding: 40px;

text-align

Adjust the align inside an element

text-align: right;
